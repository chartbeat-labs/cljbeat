Cljbeat is the collection of clojure libraries and utilities that we at Chartbeat thought would be useful to share.

The cljbeat project list

  • cljbeat-hll - Implementation of HyperLogLog probabilistic counters
  • cljbeat-opentsdb - OpenTSDB client library
  • cljbeat-options - Command line option parsing utilities

This project is maintained by chartbeat-labs
